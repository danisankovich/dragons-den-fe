Point Allocation System

A new point buy system for character creation. All stats start at 6, and you get 36 points to allocate as you see fit with the following rules:

  • No stat can be above 16 before racial bonuses and ability score improvements are applied
  • Only two stats can be at 16 before racial bonuses and ability score improvements are applied
  • You apply racial bonuses and level-up ability score improvements after point allocation is finished
